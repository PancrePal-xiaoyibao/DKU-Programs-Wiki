胃肠道间质瘤（GIST）是胃肠道最常见的间叶源性肿瘤，属于消化道间叶组织起源的肉瘤，占所有胃肠道恶性肿瘤的1%-3%。其年发病率约为（10-20）/100万，近年来因病理检测技术进步检出率逐渐升高。该病高发于中老年人群，中位发病年龄约65岁，男女发病率无明显差异。约50%-70%的病例发生于胃部，其次是小肠（20%-30%），结直肠和食管占比不足10%。其生物学行为多样，可表现为良性至恶性，约30%为恶性肿瘤，主要转移部位为肝脏和腹腔。
